如何做好文献阅读及笔记整理

一、文献阅读目的

  1. 了解背景知识——在对课题一无所知、亟待探索阶段，推荐阅读硕博学位论文第一章文献综述部分，领域大牛文献綜述（相比学位论文，内容和范围更精，更窄，适合阅读完学位论文后再来阅读）
  2. 把握课题研究方向行业——在对课题已经了解，深化认知阶段，建议阅读行业最新论文，领域大牛文献综述，了解哪些方向是尚未实施的，以确定这一课题是否切实可行，也可以阅读最新相关论文，这些论文代表该行业的最新趋势，在这一趋势上进行升华，多多尝试
  3. 学习课题设计思路——对于课题开始实践，设计试验阶段，建议阅读时不要只局限于本领域最新研究论文，去阅读其他领域前沿的论文，思考该思路能否应用于本领域，来提供新思路
  4. 提升逻辑写作能力——适用阶段：全部过程，阅读建议：阅读顶刊

二、如何筛选所需文献（缩小搜索范围，切忌盲目下载一堆论文，不做任何筛选）

首先看搜索文献的标题，如果标题与所研究的内容相关，可进一步查看关键词与研究是否匹配，如果相符合，可进一步阅读摘要部分（摘要中有：背景、目的、试验结果、结论等内容），如果摘要部分与所研究内容相关，这篇文章就是需要的关键文章；摘要需要花时间认真阅读。

常用工具：
①文献鸟 （storkapp.me）——通过输入研究方向关键词，设定提醒时间，文献鸟会定时将符号关键词的文献发送邮件，从而获得最新的相关研究文献；

②Connected Papers（Connectedpapers.com）——通过输入文献名称或者 DOI，可以获得与文献相关的论文知识图谱，追踪文献的引用和被引关系。通过它可以轻松了解到文章与文章之间的练习，有助于引领我们认识该学科的最新前沿，并帮助我们了解该领域中最重要的文献，将自己的研究与重要文献相比较，以得出相应的结论。
网格图谱需要注意的三个关键点：①论文发表时间越新颜色越深；②论文引用度越高圆圈越大；③论文之间联系越强连线越黑。
prior works所列举的是最常被引用的论文，通常是该领域的开山之作，是了解这个领域最好的敲门砖
Derivative Works所列举的是该领域新的综述或者相关研究工作，是了解领域最新进展的垫脚石

三、如何做好文献阅读

（1）了解论文整体框架结果

研究型论文：摘要、前言、材料与方法、结果与讨论、结论

  1. 摘要：文章的核心部分，简要概括本论文所有涵盖内容，包括研究背景、论文主要目的、主要实验现象和结论、论文的意义和相应价值，读完后会对文章的核心内容有一个基本认知
  2. 前言：研究背景简介，对于一些相关的概念和知识进行补充，接着解释现有研究的不足，最后在此基础上引出本文的创新点以及目的。通过阅读前言，会让我们对于论文的整体逻辑结构框架有一个清晰的认知。
  3. 材料与方法：试验材料、试验步骤、分析方法，可在这里找到应用于后续课题试验的操作指南
  4. 结果与讨论：解释试验现象与机理、他人研究对比，阅读这部分有助于思考该部分内容能否运用于未来的课题研究中，阅读作者介绍的本研究的不足之处及未来展望，可以思考是否在此基础上做一个相应的延续，去和自己的课题内容做一个结合
  5. 结论：凝练总结本论文主要试验现象和相关机理，有助于快速了解文章的核心内容

（2）针对不同的目标有选择性地阅读文献

  1. 为了了解课题背景知识。精读：摘要一结论一前言，略读：结果与讨论一材料与方法
  2. 为了把握课题研究方向。精读：摘要一结论一前言，选择性阅读：结果与结论（不足及期望），略读：材料与方法。通过与自己的课题进行比较，会对于自己课题未来的研究方向有一个认知，未来与展望部分会直观告诉你现有研究的不足以及可行性，特别是一些大牛的文章，通过阅读该部分可以对文章进行针对性的修改和变动
  3. 为了学习课题设计思路。精读：摘要一结论与结果一材料与方法，选择性阅读：前言（论文创新点与可行性分析）一结论。先读摘要部分，对于文章核心内容有一个初步的认知，由于此时已经对课题有一个深刻的认知了，所以直接阅读结果与讨论部分，特别需要关注文章的图表以及描述，可以使你快速了解文章的基本结果，同时这些结果也可以让你思考自己的试验和研究是不是也可以按照他的这种来，随后开始仔细分析结果与讨论部分的逻辑内容，以及逻辑框架结构，通过大量阅读总结出你这个方向上的研究做法的思路，最后返回到材料和方法中，了解到这些方法的试验设计和分析手段，这样即可以在后面的课题中加以运用，泛读前言和结论部分主要是为了了解论文的创新点和可行性分析。
  4. 为了提升逻辑写作能力。精读：摘要一前言，选择性阅读：结果与讨论（小标题及每段开头结尾）一结论，略读：材料与方法。摘要是我们直观了解文章的最好窗口，摘要写得如何很大程度上决定审稿人对文章的态度；其次是精读引言部分，它是最好体现文章逻辑框架的部分，因此，通过精读摘要和引言是最容易提升我们逻辑写作能力的；此外对结果与讨论部分的小标题进行仔细阅读，学习顶刊是如何在标题中体现亮点的，同时，也可以仔细阅读每篇文章的开头与结学习好的文章的每一段的衔接以及谋篇布局，通过不断地模仿和学习，有助于提高自身的逻辑写作能力。

四、如何做好笔记整理

为什么要做好笔记整理？

  1. 过往文献长时间不阅读会遗忘
  2. 学习如何阅读文章以及搭建框架，逐渐开始发现什么是重要的，什么是不重要的，可以大大提升阅读效率，同时通过拆解文章，也会对文章的架构有一个清晰的知道，对于后续的写作有很大帮助
  3. 方便写作时对比讨论，引用文献，增强论文的可读性，以及深化自己对于机理的理解

文献笔记错误认知？

  1. 文献笔记不是简单的勾画和机械的抄写，一些之前勾画过的地方以后再看也不知道原因，还是得重新阅读，所以主要是要体现一些自己对文章的思考和态度，我认为这篇文章好在哪里，我能从中借鉴到、学习到什么
  2. 文献笔记不是为了给别人看的，最终目的都是为自己服务，可以完全按照自己的偏好语言、缩写、习惯等进行记录，只要自己看得懂就行（弹幕提供的一个读文献做笔记的软件：readpaper）
  3. 文献回顾不是文献综述，我们的目的是阅读，不需要根文献综述一样前因后果面面俱到，要根据我们的目的有重点性地

如何做好笔记整理？

重点回答“文章好在哪里”；以及应用“125”原则：1个思路，2个绘图，5个句式

这样的阅读模式，将自己的阅读成果量化，强迫自己在这个过程中去思考，去学习

  1. 对于文章的信息进行简要的记录，包括作者、单位、期刊以及名称等，其中，作者和单位一般只关注第一或者通讯作者，了解哪个课题组在进行这个研究即可
  2. 我们需要列出文章的基本框架，对于研究背景、研究目的以及研究结论进行简要的摘录
  3. 结果与讨论部分，对于文章出现的关键图表，关键机理解释、重点部分等等进行相应的笔记记录，方便后续阅读，直接了解文章的重点所在
  4. 强迫自己思考这篇文章好在哪里，只有当自己理解这篇文章好在哪里，为什么能发在顶刊上，那么你也就基本理解了全篇文章的基本内容
  5. 强迫自己总结这篇文章可以借鉴学习的地方，“125”原则：1个思路，2个图表，5个句式。
    1个思路：思考这篇文章可能会给自己未来的研究带来什么思路，不管是实验方法上，还是研究思路上；
    2个绘图：学习这篇文章画的较好的图表，不管是形式还是配色；
    5个句式：总结这篇文章写得比较好的五个句式，并且尝试着模仿，把它运用到你的论文写作中来
    通过这种模式，可以将自己的阅读成果量化，强迫自己在这个过程中去思考去学习，对于个人未来的发展有着极大的裨益！
Q&A

在此关于文献笔记向up主和各位观众请教一个小问题：

1.完成一项研究大致需要阅读多少篇文章或记录多少阅读笔记；

一般来说，完成研究阅读文献是一个长期的过程，根据你课题的难易而决定，同时在这个过程你还需要参考其他的，所以了解方向的话可能10-20，真正完成研究至少50+吧

2.阅读笔记如何管理，以 up 给出的笔记为例，虽然条例很清晰简介，但这样一篇文章就需要这一页 PPT 的大小来做笔记。参考上面的问题，完成一项研究，可能需要十几篇、几十篇类似的笔记积累。请问 up 和大家用什么软件工具制作笔记并管理笔记呢；

这个我是拿word弄的，你可以试试notion，据说很好用

3.如何有效管理笔记，让笔记发挥作用。我也尝试过做一些笔记，文章卡片之类的。但发现做好的笔记和卡片反而又成了一种阅读负担。有的是做笔记不够清晰，还需要回去找出原本的文章再读一读，有的是笔记变多变杂以后，比较不同的笔记也容易不着重点，不知道该怎么进一步梳理。甚至打开笔记也和打开文献一样有压力了。

这个做的文献笔记主要是针对需要精读的文献，如果你至少泛读的话，只需要稍微记一下，同时整理的时候分清楚精读和泛读，按一些关键词进行分类就好了
