(f) 投资效率的定义及影响

定义投资效率

投资效率可以理解为企业将投资转化为生产资本的效率。具体而言，投资效率越高，企业通过单位投资可以获得的有效资本就越多。投资效率的下降意味着企业需要更多的投资才能获得同样的有效资本量。

影响企业家净资产和信用利差的因素

企业家风险的意外增加

企业家风险的增加意味着 ω\omegaω 的分布发生变化，可能的情况是 ω\omegaω 的均值降低，或者方差增加，使得企业家面临更多的不确定性和更大的违约风险。

  1. 对企业家净资产的影响：

    • 企业家面临更高的风险，意味着实际获得的资本回报率 RkR_kRk​ 可能更低。
    • 由于风险增加，企业家违约的可能性增加，这导致企业家的净资产减少。

  2. 对信用利差的影响：

    • 信用利差 Z−RZ - RZ−R 表示企业家的借款成本相对于无风险利率的溢价。
    • 随着企业家风险的增加，银行为了补偿更高的违约风险，会提高贷款利率 ZZZ，从而导致信用利差 Z−RZ - RZ−R 增加。

投资效率的意外下降

投资效率的下降意味着企业需要更多的投资才能产生同样的资本回报。假设投资效率降低，可以表示为单位投资产生的有效资本减少。

  1. 对企业家净资产的影响：

    • 投资效率下降会导致资本供给减少，资本价格上升。
    • 虽然单位投资的资本产出减少，但由于资本价格上升，企业家的总资本价值可能增加，从而增加企业家的净资产。

  2. 对信用利差的影响：

    • 随着资本价格的上升，企业家的资本价值增加，净资产增加，意味着企业家有更高的资本来作为抵押，违约风险降低。
    • 银行面临的违约风险降低，贷款利率 ZZZ 可能下降，从而导致信用利差 Z−RZ - RZ−R 减少。

投资效率变化的具体影响

  1. 企业家风险增加（ω\omegaω 分布变化）：

    • 风险增加 -> 净资产减少 -> 借款成本上升 -> 信用利差增加。
    • 这种情况下，企业家面临更高的不确定性，贷款成本上升，导致其投资和经济活动减少。

  2. 投资效率下降（单位投资的有效资本减少）：

    • 投资效率下降 -> 资本供给减少 -> 资本价格上升 -> 净资产增加。
    • 净资产增加 -> 借款成本下降 -> 信用利差减少。
    • 这种情况下，虽然单位投资的产出效率下降，但由于资本价格上升，企业家的总资本价值增加，违约风险降低，贷款成本下降。

经济直觉

  1. 企业家风险增加：

    • 当企业家面临更高的风险时，银行提高贷款利率以补偿风险，这增加了企业家的借款成本。
    • 净资产减少，导致企业家更难以获得融资，信用利差扩大，整体投资和经济活动减少。

  2. 投资效率下降：

    • 投资效率下降虽然使得单位投资的资本产出减少，但资本价格上升使得企业家的资本价值增加。
    • 净资产增加，降低了违约风险，银行降低贷款利率，信用利差缩小，企业家的融资环境改善。

总结

在BGG模型中，企业家风险的意外增加和投资效率的意外下降对企业家净资产和信用利差有相反的影响：

  • 企业家风险增加 -> 净资产减少 -> 信用利差增加。
  • 投资效率下降 -> 净资产增加 -> 信用利差减少。
